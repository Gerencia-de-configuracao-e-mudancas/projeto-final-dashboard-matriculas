# Projeto Final - Dashboard de Matrículas

Este projeto é uma atividade acadêmica para a disciplina de **Gerência de Configuração e Mudanças**, desenvolvida pelos alunos do 4º ano do IFPB. O objetivo é criar uma interface web para um sistema de matrículas, aplicando boas práticas de desenvolvimento e controle de versão com Git e GitHub.

## ✨ Funcionalidades

O projeto consiste em um painel de controle (dashboard) com as seguintes páginas e funcionalidades:

* **Dashboard Principal:** Apresenta um resumo com o total de alunos, cursos e cidades.
* **Página de Cidades:** Exibe uma tabela com a lista de cidades, permitindo as ações de "Editar" e "Remover". Inclui um botão para "Novo Cadastro".
* **Página de Cursos:** Exibe uma tabela com a lista de cursos, também com opções de "Editar", "Remover" e "Novo Cadastro".
* **Página de Alunos:** Apresenta a lista de alunos do sistema, com funcionalidades para adicionar, editar e remover registros.
* **Layout Responsivo:** A interface se adapta a diferentes tamanhos de tela (desktop, tablet e mobile).

## 🚀 Tecnologias Utilizadas

* **HTML5:** Para a estruturação semântica do conteúdo.
* **Tailwind CSS:** Um framework CSS "utility-first" para a estilização rápida e moderna da interface, utilizado via CDN.

## 📂 Estrutura de Arquivos

```
/
├── index.html      # Página principal do Dashboard
├── cidades.html    # Página com a lista de cidades
├── cursos.html     # Página com a lista de cursos
├── alunos.html     # Página com a lista de alunos
└── README.md       # Este arquivo
```

## ▶️ Como Executar o Projeto

Como este projeto utiliza apenas **HTML e CSS (via CDN)**, não há necessidade de um servidor ou processo de compilação.

1.  Clone o repositório para a sua máquina local:
    ```bash
    git clone [https://github.com/Gerencia-de-configuracao-e-mudancas/projeto-final-dashboard-matriculas.git](https://github.com/Gerencia-de-configuracao-e-mudancas/projeto-final-dashboard-matriculas.git)
    ```
2.  Navegue até a pasta do projeto.
3.  Abra qualquer um dos arquivos `.html` (por exemplo, `index.html`) diretamente no seu navegador de preferência (Google Chrome, Firefox, etc.).
